The Second Task

编程实现：将数组作为参数传递给一个方法，该方法将这个数组的元素全部显示出来。

我们最容易想到的可能就是通过循环来写，比如说这个样子 ☞

但是其实可以简单一点 ，我们可能会想到将数组转换成字符串来输出，想到就可以先试试 ☞

输出：

[I@60e53b93

很遗憾，这和我们直接System.out.println(array);的结果一样，没有得到我们想要的结果。

这是因为在Java中arrays并没有继承Object.toString()，我们得到的输出实际上是getClass().getName() + '@' + Integer.toHexString(hashCode())形式的数组的内存地址。

那么我们就可以借助Java API中提供的Arrays类，在java.util包中，更改代码：

System.out.println(Arrays.toString(array));

输出：

[1, 2, 3]

所以我们得到了比循环更简单的打印方式：Arrays.toString()。

但是问题又来了，如果我想要打印的数组不是一维数组会怎么样呢？比如说 ☞

输出：

[[Ljava.lang.String;@60e53b93, [Ljava.lang.String;@5e2de80c]

Arrays.toString()不可行了，我们换成Arrays.deepToString()

输出：

[[Why, doesn’t], [it, rain, on, Wednesday]]
